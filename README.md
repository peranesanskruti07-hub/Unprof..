# UnProf Worker Profile Card 🚀

A clean and responsive Worker Profile Card built as part of the UnProf Internship Day 1 Task using HTML and CSS.

## 📌 Project Overview
This project showcases a static worker profile card with essential details such as profile photo, profession, location, skills, and contact option. The design is user-friendly, visually appealing, and fully responsive for both mobile and desktop screens.

## ✨ Features
- Profile Photo Section
- Worker Name & Profession
- Location Display
- Skills Section
- About/Bio Section
- Contact Button
- Responsive Design

## 🛠 Technologies Used
- HTML5
- CSS3

## 🎯 Objective
The goal of this task was to strengthen fundamentals of:
- HTML Structure
- CSS Styling
- Responsive Design
- UI Development

## 📷 Output
A modern and responsive worker profile card with clean UI.
